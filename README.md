# Блок-схема программы моделирования обработки задач

## Основная программа

```mermaid
graph TD;
    A(Начало) --> B{Итерации (5 раз)}
    B -->|Да| C(Обнуление счетчиков)
    C --> D{Итерации (EXPERIMENTS раз)}
    D -->|Да| E[Вызов simulate()]
    E --> F(Обновление статистики)
    F --> D
    D -->|Нет| G(Вычисление средних значений)
    G --> H(Вывод текущих результатов)
    H --> B
    B -->|Нет| I(Вычисление итоговых средних значений)
    I --> J(Вывод итоговых значений)
    J --> K(Конец)
graph TD;
    A(Начало) --> B(Инициализация переменных)
    B --> C(Генерация времени поступления задачи)
    C --> D{Обработано TASKS_COUNT задач?}
    D -->|Нет| E(Обработка очереди задач)
    E --> F(Обновление состояния ПК)
    F --> G(Генерация новой задачи при необходимости)
    G --> H(Инкремент времени)
    H --> D
    D -->|Да| I(Подсчет итоговых значений)
    I --> J(Конец)
